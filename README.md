# Parallel programming

> [databass bookclub](databass.dev)

Rust workspace for Parallel Programming (cs149). The original assignments are C++/ISPC on Stanford's myth machines; this workspace is a Rust port done for learning purposes, program by program.

## Running

```
cargo run -p assn1-prog1
cargo run -p assn1-prog2
```

## Testing

```
cargo test
```
